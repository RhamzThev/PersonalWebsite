# Personal Website

This repository contains the source code for a sleek and modern personal website showcasing a portfolio, resume, and blog. The website is built using HTML, CSS, and JavaScript, designed to highlight skills and projects effectively.

## Features

- **Portfolio:** Showcases various projects with descriptions and images.
- **Resume:** Embedded PDF resume for easy download.
- **Blog:** Section to share thoughts and articles.
- **Contact:** Links to GitHub, LinkedIn, and email for communication.

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **FontAwesome**

## Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/RhamzThev/PersonalWebsite.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PersonalWebsite
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Usage

1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open `http://localhost:3000` in your browser to view the website.

## File Structure

- **/components**: Reusable UI components
- **/pages**: Next.js pages for each section
- **/styles**: SCSS files for styling
- **/public**: Static assets like images and fonts
